Instale a governança ARGUS v4.0.0 neste repositório.

Passos:

1. Busque o conteúdo de cada arquivo abaixo diretamente da URL raw do GitHub (raw.githubusercontent.com/talessc74/argus-governance/main/...) usando as ferramentas disponíveis:
   - CLAUDE.md
   - .seeds/ARGUS.md
   - .seeds/BAU.json
   - .seeds/BLAST.json
   - .seeds/COMPASS.json
   - .seeds/EMPIRICUS.json
   - .seeds/FLUX.json
   - .seeds/GHOST.json
   - .seeds/LITERATE.json
   - .seeds/PARETO.json
   - .seeds/POLARBEAR.json
   - .seeds/PROBE.json
   - .seeds/RIVERRAID.json
   - .seeds/SCAFFOLD.json
   - .seeds/SCOUT.json
   - .seeds/SENTINEL.json
   - .seeds/SOVEREIGN.json

2. Se já existir um CLAUDE.md neste repositório, renomeie-o para CLAUDE.md.bak antes de sobrescrever.

3. Escreva todos os arquivos no repositório atual, mantendo a estrutura de pastas (.seeds/ na raiz).

4. Faça commit com a mensagem: "chore: instalar governança ARGUS v4.0.0"

5. Faça push para o branch atual.

6. Confirme ao usuário que a governança foi instalada e que o CLAUDE.md será lido automaticamente a partir do próximo chat neste repositório.
